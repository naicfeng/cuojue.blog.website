使用方法

兼容Clash和ClashR配置文件

linux

直接替换原二进制文件即可

windows

直接替换原.exe 文件即可

如使用 GUI 客户端 clash_for_windows_pkg
可进入Clash.for.Windows\resources\static\files
替换clash-win64.exe文件，支持订阅 ClashR 配置
