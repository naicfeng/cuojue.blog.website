分享全平台ClashR Core客户端下载

官方已支持ssr协议，请使用官方版
建议弃用ssr协议

使用方法

兼容Clash和ClashR配置文件

linux

直接替换原二进制文件即可

windows

直接替换原.exe文件即可

如使用GUI客户端 clash_for_windows_pkg
可进入Clash.for.Windows\resources\static\files
替换clash-win64.exe文件，支持订阅ClashR配置

广告时间

推荐一个在各种订阅格式之间进行转换的实用程序项目 SubConverter
本站公益在线配置转换 https://cuojue.org/sub

支持的R算法

Encrypting algorithm

  • aes-128-cfb
  • aes-192-cfb
  • aes-256-cfb
  • aes-128-ctr
  • aes-192-ctr
  • aes-256-ctr
  • aes-128-ofb
  • aes-192-ofb
  • aes-256-ofb
  • des-cfb
  • bf-cfb
  • cast5-cfb
  • rc4-md5
  • chacha20
  • chacha20-ietf
  • salsa20
  • camellia-128-cfb
  • camellia-192-cfb
  • camellia-256-cfb
  • idea-cfb
  • rc2-cfb
  • seed-cfb

Obfs

  • plain
  • http_simple
  • http_post
  • random_head
  • tls1.2_ticket_auth

Protocol

  • origin
  • verify_sha1 aka. one time auth(OTA)
  • auth_sha1_v4
  • auth_aes128_md5
  • auth_aes128_sha1

ps:条件有限，仅测试 auth_sha1_v4，auth_aes128_md5 通过
